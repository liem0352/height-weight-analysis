# 身高体重数据分析

> 作者：liem

## 项目简介

本项目使用 Python 对 30 名大学生的身高与体重数据进行相关性分析，通过 Shapiro-Wilk 正态性检验判断数据分布特征，再选择合适的统计方法（Pearson 或 Spearman）分析身高与体重的相关性。

## 分析流程

```
数据准备 → 正态性检验（Shapiro-Wilk）→ 选择检验方法 → 相关性分析 → 结果解读
```

## 技术要点

### 1. 正态性检验（Shapiro-Wilk）
- 原假设：样本数据服从正态分布
- 判断标准：p > 0.05 则接受原假设

### 2. Pearson 相关分析
- 适用条件：两个变量均服从正态分布
- 衡量线性关系的强度和方向
- 相关系数 r ∈ [-1, 1]

### 3. Spearman 秩相关分析
- 适用条件：不满足正态分布或有序分类变量
- 基于秩次计算相关性
